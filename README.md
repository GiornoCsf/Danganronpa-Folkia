# Danganronpa Fangame

A Danganronpa-inspired fangame based on our Discord server and its members.

## Tech
- C++
- raylib
- CMake

## Current Status
Prototype with:
- basic window setup
- tilemap rendering
- player sprite rendering
- movement testing
